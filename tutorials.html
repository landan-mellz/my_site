1. create repository in your github account
  2. copy the link
  3. open your gitbash
  4. type in "git clone link"
  5. check your link on the files, open document
  6. on your folder make chages
  7. add changes " git add "
  8. commit your chages "git commit -m 'discription'"
  9. you git log to check your commit "git log"
  10. then "cd name of your repository"
  11. set up the username and email
    • git config --local user.name "username"
    • git config --local user.email "email"
  12. then "git push"
